Choosing an estimator
==============================

Choosing the right estimator for your causal question is crucial, and typically, not all methods are possible.
Much depends on if you can run an experiment, or if you only have access to historical data. Though there is no
single answer to "which estimator should I use?", we have created an online wizard to help you choose
on https://getformative.dev.

The wizard includes the estimators currently implemented in formative, and will be updated as new methods are added.
